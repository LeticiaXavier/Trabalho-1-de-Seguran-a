## 🚀 Recursos Principais
- Implementação completa do S-DES
- Modos de operação:
  - ✅ ECB (Electronic Codebook)
  - ✅ CBC (Cipher Block Chaining)
- Geração de subchaves (K1 e K2)
- Funções básicas:
  - Permutação P10/P8
  - Expansão E/P
  - S-Boxes (S0 e S1)
  - XOR bit-a-bit
